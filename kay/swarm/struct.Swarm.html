pub struct Swarm<SubActor> { /* fields omitted */ }

Offers efficient storage and updating of large numbers of identical SubActors and is typically used whenever there is more than one actor with the same type/behaviour.

If SubActor can receive Message, then Swarm<SubActor> can do so as well, redirecting each such message to the correct sub-actor by default (see RecipientAsSwarm)
